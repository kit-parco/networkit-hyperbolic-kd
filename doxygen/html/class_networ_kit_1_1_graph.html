Merges edge {u,v} to become a supernode.

Edges to u and v are rewired, multiple edges merged and their weights added. The vertex weights of u and v are added. A self-loop is only created if discardSelfLoop is set to false.

Returns

Remove an isolated node from the graph.

Although it would be convenient to remove all incident edges at the same time, this causes complications for dynamic applications. Therefore, removeNode is an atomic event. All incident edges need to be removed first and an exception is thrown otherwise.
